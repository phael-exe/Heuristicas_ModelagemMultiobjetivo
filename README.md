## 📚 Sobre a disciplina

A disciplina aborda tecnicas de busca, otimizacao e tomada de decisao aplicadas a problemas computacionais. O foco e estudar metodos capazes de encontrar boas solucoes para problemas em que uma busca exaustiva pode ser inviavel, alem de discutir cenarios com multiplos objetivos e criterios de avaliacao.

Entre os temas presentes no repositorio estao:

- 🔎 Agentes de busca e busca sistematica
- ♟️ Busca adversaria
- 🧭 Busca local
- 🚫 Busca tabu
- 📈 Modelagem e resolucao de problemas de otimizacao

## ▶️ Como executar

Para executar os notebooks localmente, crie um ambiente virtual e instale as dependencias:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Depois, abra os notebooks com Jupyter, VS Code ou outra ferramenta compativel com arquivos `.ipynb`.
